【运行顺序】
载入工作空间report
在input.r首两行更改id（1-25）
运行想看的display脚本观看结果


【脚本功能】
input
#############################
功能：输入id进行伪gps定位工作确定大聚落



display_mapsurface
#############################
依赖包：leaflet
功能：显示筛选完后大聚落中所有店，以小聚落上色



display_3dplot
#############################
依赖包：plotly
功能：显示大聚落中餐馆Value（以每点中心的窗内总星级）


display_top3
#############################
依赖包：leaflet
功能：显示筛选完后最高分三个小聚落所有店，寻找三个中心点和半径，绘制范围，再讲测试点绘制在地图上


【注意】leaflet画范围这里用的不是绝对范围语句，用绝对距离那个函数不停报错，暂时只用相对半径函数（如同画一个标记,半径随缩放倍率变动）演示一下
